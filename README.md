## Escenario
Escenario A – Análisis de Datos Climáticos

## Dataset utilizado
Dataset simulado de registros mensuales de temperatura máxima, temperatura mínima 
y precipitaciones durante el año 2023. Almacenado en `/datos/clima_2023.csv`.

## Estructura del repositorio
- `/datos` → Dataset CSV con registros climáticos
- `/scripts` → Script Python de análisis
- `/resultados` → Gráfico generado por el análisis

## Instrucciones para ejecutar el script
1. Clonar el repositorio
2. Abrir Google Colab
3. Ejecutar `scripts/analisis_climatico.py`
4. Los resultados se guardan automáticamente en `/resultados`

## Gestión del proyecto
Las tareas fueron organizadas en Jira bajo el proyecto Celula-Analisis-Climatico (CAC).
